## 1. 設置場所の準備

GORDIXの設置には、本体サイズに加えて前後左右に作業スペースが必要です。

- 水平で安定した床面（推奨：コンクリート土間または補強された木床）
- 100V電源（専用回路推奨）
- 集塵機の設置スペースと排気経路

## 2. 開梱と組み立て

1. 木枠を外し、本体を設置位置まで移動する
2. 固定用ボルトを外し、水平を調整する
3. スピンドル・集塵ホース・制御PCを接続する

## 3. ソフトウェアのインストール

制御用PCに専用CAMソフトをインストールします。

```text
1. 付属USBメモリからインストーラを起動
2. 画面の指示に従いインストール
3. 機体シリアル番号を入力してライセンスを有効化
```

## 4. 初回の試し切削

付属のサンプルデータ（コースター）を使って、初回の切削を行います。
切削中は必ずカバーを閉め、非常停止ボタンの位置を確認してから開始してください。
