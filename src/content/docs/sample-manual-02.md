## 用意するもの

| 品目 | 目安 |
| --- | --- |
| シナベニヤ | 9mm厚 300×300mm |
| エンドミル | 直径4mm 1枚刃 |
| 両面テープ | 材料固定用 |

## 手順の流れ

1. **データをつくる** — CADソフトで直径90mmの円と模様を描く
2. **CAMで経路を出す** — 切削順序・送り速度・切込み深さを設定する
3. **材料を固定する** — 捨て板の上に両面テープでしっかり固定する
4. **切削する** — 原点合わせをして加工を開始する

## つまずきやすいポイント

- 材料の固定が甘いと、切削中にズレて刃が折れることがあります
- 送り速度が速すぎると焦げ、遅すぎると刃に負担がかかります
- まずは付属のプリセット値から始めるのがおすすめです
